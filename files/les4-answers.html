Vraag 15
Wat is het verschil tussen een array als parameter en een primitief type (zoals int) als parameter?
Bij een array wordt de verwijzing gekopieerd (pass-by-reference), bij primitieve types wordt de waarde gekopieerd (pass-by-value). Wijzigingen aan array elementen zijn zichtbaar buiten de methode, bij primitieve types niet.

Vraag 35
Wat betekent het als een array "pass-by-reference" is?
Dat wanneer je een array als parameter meegeeft, de methode werkt met dezelfde array in het geheugen, niet met een kopie. Wijzigingen zijn dus permanent.

Vraag 39
Wat is het verschil tussen een normale for-loop en een enhanced for-loop bij arrays?
Een normale for-loop geeft toegang tot de index, een enhanced for-loop (for-each) doorloopt alleen de waarden zonder index. Enhanced for-loop is handiger als je de index niet nodig hebt.
